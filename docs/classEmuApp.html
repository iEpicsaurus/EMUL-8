Detailed Description

Links the core class of the emulator and wxWidgets.

Links between the core class of the Emulator and WxWidgets. Stores the Frame (Where the core renders to), Stores the Panel (Where the OS can create the window), and stores the Core (Where data is augmented and where the emulation of a single ROM occurs)
